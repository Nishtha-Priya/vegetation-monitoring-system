Project Setup Instructions

1. Install Python 3.11 or above.

2. Open the project folder in VS Code or Terminal.

3. Create a virtual environment:

   python -m venv .venv

4. Activate the virtual environment:

   Windows:
   .venv\Scripts\activate

5. Install the required libraries:

   pip install -r requirements.txt

6. Run the Flask application:

   python app.py

7. Open the following URL in the browser:

   http://127.0.0.1:5000

Sample satellite image pairs have been provided in the sample_images folder for testing.
